# 실행용 최소 소스 구성설명

## 목적

이 문서는 `팀공유_수어MVP_실행용최소소스_2026-04-24.zip` 안에 포함된 파일과 폴더가 왜 들어 있는지 설명하는 안내 문서임.

## 포함 목적

이 압축 파일은 프로젝트를 다시 실행하거나, 선택 라벨 기준으로 target sample을 다시 만들고, subset manifest를 기준으로 전처리/학습을 재현할 수 있도록 필요한 최소 소스만 모아둔 실행용 묶음임.

설명용 문서 전체는 별도 압축 파일에 있고, 이 압축 파일은 실제 실행 중심으로 구성했음.

## 포함 파일 설명

- `config/default.yaml`
  - 프로젝트 기본 설정 파일임
  - 데이터 경로, sequence length, checkpoint 경로 등 실행 기본값이 들어 있음

- `requirements.txt`
  - 공통 Python 패키지 설치 기준 파일임

- `src/data`
  - morpheme 라벨 추출, subset 생성, 전처리 관련 코드가 들어 있음

- `src/models`
  - baseline 및 sequence 모델 학습 코드가 들어 있음

- `src/inference`
  - 추론 및 실시간 추론 관련 코드가 들어 있음

- `src/services`
  - STT/TTS 보조 서비스 코드가 들어 있음

- `src/utils`
  - 공통 유틸리티 코드가 들어 있음

- `src/ui/app.py`
  - Streamlit UI 메인 파일임

- `scripts/export_selected_targets.py`
  - `selected_labels_small.json` 기준으로 `selected_label_targets.csv` 를 다시 생성할 때 쓰는 스크립트임

- `scripts/validate_subset_manifest.py`
  - subset manifest 검증용 스크립트임

- `scripts/merge_subset_manifests.py`
  - 여러 작업자 결과 manifest 병합용 스크립트임

- `data/selected_labels_small.json`
  - 현재 MVP에서 실제 사용할 최종 라벨 목록임

- `data/selected_label_targets.csv`
  - 최종 라벨에 해당하는 실제 sample id 목록임

- `data/sample_subset_manifest.csv`
  - 실제 추출된 subset 결과 manifest 임

- `data/sample_subset_manifest_template.csv`
  - manifest 작성 형식 기준 템플릿임

## 실행 흐름 요약

```text
selected_labels_small.json 수정
-> export_selected_targets.py 실행
-> selected_label_targets.csv 재생성
-> subset 추출/병합
-> sample_subset_manifest.csv 확인
-> preprocess / train / inference 진행
```

## 주의

- 이 압축 파일은 실행에 필요한 최소 묶음이므로, 전체 설명은 별도 문서 zip을 함께 보는 것이 좋음
- Python 환경 정보는 `report/python_실행환경_공유.md` 문서를 참고하면 됨

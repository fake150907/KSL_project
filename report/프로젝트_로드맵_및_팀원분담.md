## 🎯 핵심 목표 및 범위

### 목표
- 청각장애인과 일반인 간 기초적 일상대화 지원
- 웹캠 기반 수화 → 텍스트 → 음성 변환
- 음성 → 텍스트 시각화

### 인식 대상 수화 (8개 라벨)
```
가다      감사      괜찮다    배고프다
병원      아프다    우유      자다
```

### 제외 범위 (MVP 단계)
- **SEN 데이터** (문장 수준의 복합 수어)
  - WORD: 단어 단위 vs **SEN: 여러 단어의 연속된 문장**
  - MVP에서는 단어 인식만 (문장 번역 X)
  
- **CROWD 데이터** (군중 장면의 수어)
  - 여러 사람이 함께 있는 환경에서의 수어
  - 배경 복잡도, 가려진 신체 부위 등으로 처리 어려움
  - MVP에서는 1인 명확한 환경만 대상
  
- **SYN 데이터** (합성 데이터, Synthetic)
  - 실제 촬영 데이터가 아닌 인공적으로 생성한 가상 데이터
  - 3D 모델링이나 애니메이션으로 만든 수어 영상
  - MVP에서는 실제 사람의 촬영 데이터(REAL)만 사용
  
- 고도화된 비수지 신호 모델링 (얼굴 표정, 상체 움직임 등 세밀한 신호)
  - MVP에서는 손 랜드마크 중심 (얼굴 제외)
  
- 모바일 앱 (PC 우선)
  - PC기반으로 대화하는 UX로 구축됨

# MediaPipe를 활용한 수화 통역기

## 1. 프로젝트 개요
본 프로젝트는 청각장애인(농인)과 일반인 사이의 의사소통을 지원하기 위한 **양방향 통역 보조 시스템**을 개발하는 것을 목표로 한다. 시스템은 두 가지 핵심 기능으로 구성된다.

첫째, 청각장애인의 수화를 웹캠으로 입력받아 **텍스트로 변환**하고, 이를 다시 **TTS(Text-to-Speech)** 엔진으로 전달하여 음성으로 출력한다. 둘째, 일반인의 음성을 마이크로 입력받아 **STT(Speech-to-Text)** 엔진을 통해 텍스트로 변환한 뒤 화면에 시각적으로 표시한다.

특히 본 프로젝트는 **MediaPipe**를 활용하여 손의 랜드마크를 실시간으로 추출하고, 이를 기반으로 수화 동작을 인식하는 데 초점을 둔다. 최종적으로는 3주 이내에 구현 가능한 최소 기능 제품(MVP)을 완성하여, 실제 시연이 가능한 수준의 의사소통 보조 시스템을 구축하고자 한다.

---

## 2. 추진 배경 및 필요성
청각장애인은 일상생활 속 다양한 상황에서 의사소통의 제약을 겪는다. 예를 들어 음식 주문, 병원 접수, 공공기관 민원 처리, 상담, 길 안내 등 음성 중심 환경에서는 의사 전달이 원활하지 않은 경우가 많다. 또한 일반인이 수화를 알지 못하는 경우가 대부분이기 때문에 상호 간 소통 장벽은 더욱 커질 수 있다.

이러한 문제를 해결하기 위해서는 **수화 → 텍스트 → 음성 출력**, **음성 → 텍스트 시각화**의 양방향 지원이 가능한 시스템이 필요하다. 본 프로젝트는 이러한 실질적 문제를 완화할 수 있는 기술적 보조 도구를 제공하는 데 의의가 있다.

- 한국어 STT/TTS 엔진을 활용한 양방향 의사소통 보조 시스템 구축
- 실제 발표 및 시연이 가능한 수준의 프로토타입 완성

## 4. 시스템 구성
본 시스템은 크게 두 가지 방향의 통역 기능으로 구성된다.

### 4.1 청각장애인 → 일반인
1. 사용자가 카메라 앞에서 수화를 수행한다.
2. 웹캠 영상이 입력되면 OpenCV를 통해 프레임 단위로 처리한다.
3. MediaPipe가 손의 위치, 손가락 관절 좌표 등 랜드마크를 추출한다.
4. 추출된 특징값을 기반으로 학습된 분류 모델이 수화 의미를 판별한다.
5. 판별된 결과는 텍스트로 변환된다.
6. 변환된 텍스트는 TTS 엔진을 통해 음성으로 출력된다.

### 4.2 일반인 → 청각장애인
1. 일반인이 마이크에 대고 음성을 입력한다.
2. 입력된 음성은 실시간 또는 준실시간으로 수집된다.
3. 음성 구간 감지 후 STT 엔진으로 전달한다.
4. STT 엔진이 음성을 텍스트로 변환한다.
5. 변환된 문장은 화면에 시각적으로 표시되어 청각장애인이 확인할 수 있도록 한다.

## 5. 세부 구현 방안

### 5.1 수화 인식 파트
수화 인식은 웹캠으로 입력된 영상을 기반으로 동작한다. OpenCV를 통해 프레임을 처리한 뒤, MediaPipe를 사용하여 손 랜드마크를 추출한다. 이때 손가락 관절 좌표, 손의 상대적 위치, 프레임 간 움직임 변화 등을 특징값으로 활용한다.

실제 수화는 손동작뿐 아니라 몸의 방향, 상체 움직임, 얼굴 표정 등 다양한 요소를 포함하지만, 본 프로젝트는 3주라는 제한된 기간 내 구현 가능성을 고려하여 **손동작 중심의 인식**에 집중한다. 따라서 우선적으로 손 모양과 움직임 기반의 생활 표현 수화를 분류 대상으로 설정한다.

### 5.2 STT 파트
STT는 일반인의 음성을 텍스트로 변환하는 기능으로, 본 프로젝트의 완성도를 좌우하는 핵심 요소 중 하나이다.

구현 흐름은 다음과 같다.

- 마이크 입력 수집
- 음성 구간 감지
- STT API 호출
- 텍스트 결과 출력

본 프로젝트에서는 한국어 인식 성능과 구현 편의성을 고려하여 **Google Cloud Speech-to-Text**를 활용한다.

### 5.3 TTS 파트
TTS는 수화 인식 결과를 일반인에게 자연스럽게 전달하기 위한 기능이다. 분류 모델이 예측한 텍스트를 음성으로 변환하여 출력하며, 시스템의 실사용성과 시연 효과를 높이는 데 중요한 역할을 한다.

본 프로젝트에서는 **Google Cloud Text-to-Speech**를 활용하여 한국어 음성을 생성한다.
